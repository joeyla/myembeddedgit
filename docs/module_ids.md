
# Module ID Allocation

| Range        | Area                              |
|--------------|-----------------------------------|
| 0x100–0x1FF | core (utilities, interfaces)      |
| 0x200–0x2FF | services (middleware)             |
| 0x300–0x3FF | platform/mcu bindings             |
| 0x400–0x4FF | platform/boards                   |
| 0x500–0x5FF | platform/os                       |
| 0x600–0x6FF | apps                               |
